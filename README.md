# ⚙️ Instalação

```bash
# Clone o repositório
$ git clone https://github.com/Mahflima/Rocket-Coffee.git

# Entre no repositório
$ cd Rocket-Coffee

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run dev
```
